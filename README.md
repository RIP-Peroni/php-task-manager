### Requirements
* PHP ^8.0.0
* Composer v2.0
* Node.js & npm
* MySQL for local

### Setup
```
git  clone https://github.com/RIP-Peroni/php-project-lvl4.git
```
```
cd php-project-lvl4
```
```
make setup
```
```
make start
```
*open browser with url:* <br>
```
localhost:8000
```
